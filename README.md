# Finance & Supply-Chain Analytics
## Project Overview
This project focuses on Finance and Supply Chain Analytics for AtliQ Hardware, leveraging advanced SQL techniques to generate actionable business insights. The analysis covers sales performance, customer insights, financial reporting, and demand forecasting accuracy to support data-driven decision-making.

## Objectives
- Perform advanced SQL analysis on sales and supply chain data
- Build financial reports for business performance tracking
- Analyze customer, product, and market performance
- Evaluate forecasting accuracy for supply chain optimization

## Tools & Technologies
- [x] SQL (MySQL)
- [x] Advanced SQL Concepts:
> - CTEs
> - Views
> - Stored Procedures
> - User Defined Functions (UDFs)
> - Joins & Window Functions
> - Subqueries & Temporary Tables
